Algoritmo CalcularCircunferencia

Início
    // Declaração de variáveis
    Real: d, A, C
    Constante Real: PI ← 3.1416

    // Entrada de dados
    Escreva("Qual é o raio da sua circunferência? ")
    Leia(d)

    // Processamento
    A ← PI * (d ^ 2)
    C ← 2 * PI * d

    // Saída de dados
    Escreva("A área: ", A)
    Escreva("O comprimento: ", C)

Fim
